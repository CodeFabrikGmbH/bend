/webhooks/stripe

Regeln, was bend mit eingehenden Requests auf diesem Pfad macht — durchleiten oder eine Mock-Antwort liefern.

Reguläre Ausdrücke erlaubt, z. B. /api/aggregate/\d{4}-\d{2}-\d{2}
Ist ein Ziel gesetzt, leitet bend weiter. Ist es leer, wird die Mock-Antwort unten zurückgegeben.

Was ist bend?

Ein Request-Catcher & Mock-Proxy. Zeig einen beliebigen Webhook oder Client auf deine Catch-URL — bend fängt jeden Request ab, zeigt ihn live an und kann ihn an dein Ziel weiterleiten.

So funktioniert's

1 · Kopiere die Catch-URL oben und trage sie als Webhook-Ziel ein.
2 · Eingehende Requests erscheinen live in der Requests-Ansicht.
3 · Öffne einen Request, um Header und Body zu inspizieren.
4 · Über „An Ziel senden" leitest du ihn an deinen Dev-Rechner weiter.
5 · Unter „Endpoints" definierst du Mock-Antworten oder feste Weiterleitungsziele.
